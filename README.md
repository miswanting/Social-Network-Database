## 命令

### exit/quit

退出程序。

### save

保存数据库。

save [filename]

### show

显示信息。

show [all/json/person/item]

### add

新增。

add [person/man/woman/item/value]

### remove

移除。

remove [person/item/hash]

### change

修改。

change [item]

### choose

choose [person/item]
